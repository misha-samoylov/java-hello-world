# java-hello-world

Simple project to learn java.

## Install Java

1. Install Java Runtime Environment (JRE). Run java programs
2. Install Java Development Kit (JDK). Compile java code

```shell
# apt-get install default-jdk default-jre
```

Run to show version Java:

```shell
$ java -version
```

This project uses: `openjdk version "21.0.8"`

## Compile Java file

Run in terminal:

```shell
$ javac HelloWorld.java
```

That is create compiled to byte code file `HelloWorld.class`.

## Run compiled file

Run in terminal compiled file `HelloWorld.class`:

```shell
$ java HelloWorld
```

Command must show result in terminal:

```shell
Hello, World!
```
